# NfQ-BoT
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/mrravanan/kannapi-2.0.git
> cd kannapi-2.0
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```
